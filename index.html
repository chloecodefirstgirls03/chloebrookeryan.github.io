About This Website

Website building is not something that came naturally to me. In fact, the my first experience of computer programming was a module that I failed. However, I have attempted to learn from my mistakes and develop my own website. This is the result!

The First Steps

I was off to a slow start, and the skeleton of my website was probably the hardest thing to create. However, with a lot of help from Google I created an outline and started to gain confidence. I added CSS, linked pages, embedded Google forms and Youtube Videos, and added some JavaScript.

The Next Steps

Next I would like to set up Google Analytics, improve the style and formatting of this website and add more JavaScript so that I can add animations.
